Tipps:
- Achtet immer darauf das ihr im richtigen directory (Ordner) seid und dass das Objekt (file oder directory) auch in eurem directory ist.
- Manche commands arbeiten mit files ("cat file.txt", würde den Inhalt des files in die Konsole ausgeben) andere arbeiten nur mit directories ( "cd /nas/name/ex1" ) 
- wichtiger Befehl: "cd ." Ändert eure directory zur aktuellen directory. Hier macht der Befehl also nichts. Den Punkt kann man aber benutzen, wenn man nicht den ganzen Pfad angeben will sondern nur von dem Ordner aus in dem ihr euch befindet. 
- viele Befehle können eingeschränkt oder spezifiziert werden. "rm" kann nur files löschen "rm -r" kann man auch directories löschen
